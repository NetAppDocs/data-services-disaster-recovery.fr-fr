= Conclusion
:allow-uri-read: 


[role="lead"]
NetApp Disaster Recovery fournit une solution DR cloud hybride économique pour votre infrastructure VMware sur site lorsqu'elle est configurée pour utiliser Amazon EVS et Amazon FSx for NetApp ONTAP comme cible DR.  Cela vous fournit une cible DR robuste pour les infrastructures critiques.  NetApp Disaster Recovery s'appuie sur les fonctionnalités natives de protection des données ONTAP qui s'appuient sur des décennies d'expérience en matière de protection des données et qui ont été testées en profondeur pour garantir que vos données sont entièrement protégées.

Pour plus d'informations :

* https://docs.aws.amazon.com/fsx/latest/ONTAPGuide/getting-started.html["Premiers pas avec Amazon FSx for NetApp ONTAP"^]
* https://docs.netapp.com/us-en/bluexp-disaster-recovery/index.html["Documentation sur la NetApp Disaster Recovery"]
